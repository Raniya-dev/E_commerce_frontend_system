# MERN E-Commerce Frontend

A responsive React-based E-Commerce application developed as part of a MERN Stack assignment. The frontend communicates with a Node.js + Express backend to provide authentication, product browsing, shopping cart, user profile management, and order placement.

---

## Features

- User Signup & Login
- JWT Authentication
- Protected Routes
- Product Listing
- Product Details Page
- Search Products
- Category Filtering
- Product Sorting
- Shopping Cart
- User Profile
- Contact Form with Validation
- Responsive UI using Tailwind CSS
- React Router Navigation
- Redux Toolkit State Management
- Axios API Integration

## Installation

### Clone Repository

```bash
git clone https://github.com/Raniya-dev/E_commerce_frontend_system.git
```

### Install Packages

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file.

```env
VITE_API_URL=http://localhost:5000
```

### Start Development Server

```bash
npm run dev
```
